## Overview

XRayMind is an end-to-end AI-powered chest X-ray analysis system that detects 14 thoracic pathologies, generates Grad-CAM attention maps, and produces structured radiology reports using LLaMA. Built as a final year research project at North South University, Bangladesh.

**Live demo:** [jalalab.github.io/XRayMind](https://jalalab.github.io/XRayMind)

---

## Features

- **14-Disease Detection** — Atelectasis, Cardiomegaly, Effusion, Infiltration, Mass, Nodule, Pneumonia, Pneumothorax, Consolidation, Edema, Emphysema, Fibrosis, Pleural Thickening, Hernia
- **Grad-CAM Visualization** — Heatmap overlay showing which regions the model focused on
- **AI Radiology Report** — Structured clinical report generated by LLaMA via Groq API
- **Web Interface** — Frontend on GitHub Pages, backend on HuggingFace Spaces

## How It Works

1. User uploads a chest X-ray on the web interface
2. DenseNet121 outputs 14 disease probabilities
3. Grad-CAM generates an attention heatmap for the top finding
4. LLaMA generates a structured radiology report (Findings, Impression, Severity, Recommendations)
5. Results displayed on the frontend

---

## Limitations

- Bias towards Infiltration due to class imbalance in NIH dataset
- Suboptimal performance on large pleural effusions
- NIH labels extracted via NLP — ~10-20% label noise
- Not validated on non-NIH datasets
