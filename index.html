// How it works

End-to-End Pipeline

Three AI systems working in sequence to analyze a chest X-ray and produce a clinical-grade report.

01 — CLASSIFY 🧠
DenseNet121 Classifier

CheXNet-style architecture trained on NIH ChestX-ray14. Outputs probability scores for 14 pathologies including Atelectasis, Pneumonia, Effusion, and more.

02 — VISUALIZE 🔥
Grad-CAM Heatmap

Gradient-weighted Class Activation Mapping highlights which regions of the X-ray the model focused on for its top prediction — making AI decisions interpretable.

03 — REPORT 📋
LLaMA 4 Scout Report

Meta's LLaMA 4 Scout (17B) receives the X-ray image and AI predictions, then generates a structured radiology report with Findings, Impression, Severity, and Recommendations.

// About the project

Built in 3 Weeks

XRayMind is a complete end-to-end medical AI system — from raw dataset to deployed web application — built as a university research project at North South University, Bangladesh.

The project demonstrates that research-grade AI (CheXNet architecture) can be replicated, improved upon, and wrapped into a usable product with modern tools like Groq's LLaMA API and HuggingFace Spaces.

Notable engineering decisions include patient-level train/val/test splitting to prevent data leakage, differential learning rates for fine-tuning, and a manual Infiltration bias correction based on dataset analysis.
